配置 Bookmarks Sync

同步平台

GitHub 认证

在 GitHub Settings 中创建 Personal Access Token， 需要勾选 gist 权限

Gitee 认证

在 Gitee 个人设置 中创建 Personal Access Token， 需要勾选 gist 权限

Gist 管理

留空则首次同步时自动创建。如果已有 Gist，可在此输入 ID。
📖 如何找到 Gist ID 如何找到 Gist ID： • GitHub：在 Gist 页面 URL 中，如 https://gist.github.com/username/123456789，ID 为 123456789 • Gitee：在 Gist 页面 URL 中，如 https://gitee.com/username/gists/123456789，ID 为 123456789
